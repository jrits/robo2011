【はじめに】
本アプリケーションは、ETロボコンのロボットからBluetoorh経由で取得したログ(csv
ファイル)を、わかりやすく可視化するツールです。

動作には、.NET Framework 4.0が必要です(Visual Studio 2010のC#で開発されていま
す)。
導入されていない場合、以下から導入してください。
http://msdn.microsoft.com/ja-jp/netframework/ff687189.aspx

X、Y、Angleの各パラメータにsvファイルの各カラムを割り当てることで、可視化を行
います。

ソースコードもあるので、バグフィックス、機能強化、魔改造などなどお好きにいじっ
てくださって結構です。

【起動方法】
Start GpsVisualizer.batくりっく
GpsVisualizer\bin\Release\GpsVisualizer.exe直接起動でもOKよ
